## Overview

PadelPro is a complete web platform for Tunisia's premier padel sports club. It features a dark-themed, responsive frontend with a REST API backend for court browsing, real-time booking with promo code integration, and SMTP-powered contact form delivery.

## Features

- **Dynamic Court Catalog** — Browse 7 professional courts with type/price filtering and detail modals
- **Real-Time Booking System** — Multi-step reservation form with live price calculation, time conflict detection, and promo code validation
- **SMTP Contact Form** — Messages delivered via Gmail SMTP as branded HTML emails with Reply-To support
- **Promo Code Engine** — Validated per-user codes with discount percentages (generated by Court Survivor mini-game)
- **Responsive Design** — Mobile-first layout with hamburger navigation, CSS grid, and media queries (1024px / 768px / 480px)
- **Scroll Animations** — IntersectionObserver-driven reveal effects for cards, sections, and content blocks
- **Toast Notification System** — Non-blocking success/error/warning feedback

## Prerequisites

- [WAMP](https://www.wampserver.com/) or [XAMPP](https://www.apachefriends.org/) (Apache + PHP 8+ + MySQL)
- [Composer](https://getcomposer.org/) (PHP dependency manager)
- A Gmail account with [App Password](https://support.google.com/accounts/answer/185833) enabled

## Installation

### 1. Clone the project

```bash
git clone https://github.com/your-username/PadelPro.git
cd PadelPro
```

### 2. Install PHP dependencies

```bash
composer install
```

### 3. Configure environment

Copy `.env.example` to `.env` and fill in your credentials:

```env
DB_HOST = "localhost"
DB_NAME = "PadelPro"
DB_USER = "root"
DB_PWD = "your-password"
DB_PORT = 3306

SMTP_HOST = "smtp.gmail.com"
SMTP_PORT = 587
SMTP_USER = "your-email@gmail.com"
SMTP_PWD = "your-app-password"
SMTP_FROM_NAME = "PadelPro"
SMTP_TO = "your-email@gmail.com"
```

### 4. Create the database

Import the SQL schema into MySQL via phpMyAdmin or CLI:

```sql
CREATE DATABASE IF NOT EXISTS PadelPro;
USE PadelPro;

CREATE TABLE clients (
    client_id INT AUTO_INCREMENT PRIMARY KEY,
    first_name VARCHAR(100) NOT NULL,
    last_name VARCHAR(100) NOT NULL,
    email VARCHAR(255) NOT NULL UNIQUE,
    phone VARCHAR(20)
);

CREATE TABLE courts (
    court_id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    surface_type ENUM('Indoor','Outdoor','Panoramic') NOT NULL,
    hourly_rate DECIMAL(10,2) NOT NULL,
    is_active BOOLEAN DEFAULT 1
);

CREATE TABLE promo_codes (
    promo_id INT AUTO_INCREMENT PRIMARY KEY,
    code VARCHAR(50) NOT NULL UNIQUE,
    discount_percent FLOAT NOT NULL,
    winner_email VARCHAR(255) NOT NULL,
    is_used BOOLEAN DEFAULT 0
);

CREATE TABLE reservations (
    reservation_id INT AUTO_INCREMENT PRIMARY KEY,
    client_id INT NOT NULL,
    court_id INT NOT NULL,
    promo_id INT DEFAULT NULL,
    booking_date DATE NOT NULL,
    start_time TIME NOT NULL,
    end_time TIME NOT NULL,
    total_price DECIMAL(10,2) NOT NULL,
    status ENUM('Pending','Confirmed','Cancelled') DEFAULT 'Pending',
    FOREIGN KEY (client_id) REFERENCES clients(client_id),
    FOREIGN KEY (court_id) REFERENCES courts(court_id),
    FOREIGN KEY (promo_id) REFERENCES promo_codes(promo_id)
);

CREATE TABLE admins (
    admin_id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(100) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL
);

-- Sample courts

```

### 5. Start the server

Place the project in your WAMP `www/` or XAMPP `htdocs/` directory, start Apache + MySQL, and visit:

```
http://localhost/PadelPro/
```

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/courts.php` | List all active courts |
| `GET` | `/api/court-detail.php?id=N` | Court details + upcoming bookings |
| `GET` | `/api/check-availability.php?court_id&date&start&end` | Check time slot availability |
| `POST` | `/api/reservations.php` | Create a new reservation |
| `POST` | `/api/promo.php` | Validate a promo code |
| `POST` | `/api/contact.php` | Send contact email via SMTP |

## Security

- **SQL Injection Prevention** — All queries use PDO prepared statements
- **Input Validation** — Client-side (JS) + server-side (PHP) dual validation
- **Environment Variables** — Credentials stored in `.env`, excluded from VCS
- **Error Handling** — Global exception/error handlers return JSON, never raw PHP errors
- **CORS Headers** — Configured in `config.php`
- **SMTP Security** — TLS encryption, App Password authentication
